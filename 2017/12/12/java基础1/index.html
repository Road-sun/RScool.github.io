数据类型

数据类型 大小位 可表示的范围
byte 8 -128~127
short 16 -32768~32767
int 32 -2147483648~2147483647
long 64 很长
float 32 -3.4E38~3.4E38
double 64 -1.7E308~1.7E308
char 16 0~255
boolean — true or false

  • 整数默认类型：int

    eg:long lg = 10000000000l;后面要加一个l，不然系统会保存因为默认为int就超出范围了。

  • 小数默认类型：double

    自动类型转换：

    基本数据类型转换遵循的原则：

    1. 容量小的类型自动转换成容量大的数据类型，数据类型按照容量大小排序byte,short,char<int<long<float<double; 注意这是是容量大小不是所占字节数。

    2. byte,short类型数据都不支持加减运算，会自动转化成int型。

    3. 容量大的数据类型转换成容量小的数据类型时，要加上强制转换符，但是可能造成精度降低，或者溢出。

    4. 有多种数据类型混合运算是，系统将自动所有数据转换成最大的那个类型，然后进行运算。
