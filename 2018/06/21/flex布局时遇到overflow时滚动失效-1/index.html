flex布局时遇到overflow时滚动失效

css
 2018/06/21

###使用flex布局时，有些需要滚动显示全部内容的元素不能滚动，其原因在于overflow失效。

解决方法 ： 如果要超出范围，需要滚动条出现，最好display:block,因为flex是柔性盒子布局，所有的子元素都会被父元素装下，甚至可能自动缩小子元素的大小


  如果一定要使用flex布局且出现滚动条，必要设置父元素和子元素的宽高，
不使用overflow:auto，使用overflow:scroll

flex-grow的使用

flex-grow 属性用于设置或检索弹性盒子的扩展比率。(父元素的多余空白空间会以子元素flex-grow的值比例分配)

如果要使子元素换行，则需要flex-warp:warp
