vue-cli搭建

  • 安装webpack
    • 全局安装 npm install webpack -g
    • 查看版本 webpack -v
    • 安装webpack-cli依赖 npm install webpack webpack-cli -g
  • 安装vue-cli
    • 全局安装 npm install –global vue-cli
    • 查看版本 vue -V

  • 用vue-cli来构建项目

    • 新建一个文件夹用来存放项目 ，cd到该文件下

    • vue init webpack xxx xxx为自定义的项目名称

    • cd到xxx文件夹下，安装依赖npm install

  • 启动项目
    • 启动项目npm run dev
    • 打包项目npm run build
    • 修改端口 config/index.js
    • assetsPublicPath 应该为’./‘ 这样打包后生成的dist文件路径才能正确
    • npm run dev后,项目会在localhost:8075上显示,如果要关闭服务器,Ctrl+C退出,如果用的git bash,则需要在任务管理器中终结node进程。
    • 生成的dist可以直接放入到服务器上
